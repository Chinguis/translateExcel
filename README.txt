How to use:

1. Get an OpenAI API key: https://platform.openai.com/docs/quickstart

2. Install Python on your computer: https://www.python.org/downloads/

3. Install packages: openai, openpyxl, asyncio

4. Put your Excel file in the same folder as this README

5. Run "python translateExcel.py"

This will create a new Excel file in the same folder, containing the translated version.
